# Minimal Portfolio Website

This is a simple single-page portfolio website for the Web Development Project 1 assignment.

## Files

- `index.html` — page structure and text
- `style.css` — black-and-white minimalist styling
- `submission_answers.md` — reflection answers to paste into the project document

## Before submitting

1. Replace placeholder links in the Contact section:
   - `https://github.com/YOUR-USERNAME`
   - `https://www.linkedin.com/in/YOUR-LINKEDIN/`
   - `your.email@example.com`

2. Deploy through GitHub Pages.

3. Paste the GitHub Pages URL into the project document.

4. Download/export the project document as a PDF and submit it.
